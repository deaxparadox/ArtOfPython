# Tuple Creation

A tuple consists of a number of values separated by commas:

```python
>>> t = 12345, 54321, 'hello!'
>>> print(t)
>>> print(type(t))
```

## Nexted Tuple

```python
>>> # Tuples may be nested:
>>> u = t, (1, 2, 3, 4, 5)
>>> u
((12345, 54321, 'hello!'), (1, 2, 3, 4, 5))
```
